This project is a springboot project created using spring initializer website.
Selected dependencies are: Freemarker, JPA, Web, MySql, Lombok
The project use Vue js for front end.

This application creates a Product table to keep product information. The SPA does CRUD operations using REST Api.

================ PART1 ================
This project is mainly created to be deployed in containers. Application and database to go in to separate containers.
The first version of the project is developed and run as simple springboot app using local database.
